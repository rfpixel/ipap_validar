# Validador de Inscripciones IPAP

Sistema desarrollado en Python con Streamlit para la auditoría y generación de planillas.

## Cómo ejecutar
1. Clonar el repositorio.
2. Ejecutar `ejecutar_proyecto_final.bat` (se encargará de crear el venv e instalar dependencias).
3. Cargar el CSV oficial en la interfaz web.
